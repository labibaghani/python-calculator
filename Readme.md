# 🧮 Python Calculator

A simple command-line calculator built with Python.

## ✨ Features

- Addition
- Subtraction
- Multiplication
- Division
- Power
- Modulus
- Input validation
- Division by zero handling
- Calculation history
- Saves history to `history.txt`
- Loads previous history when the program starts

## 📂 Project Structure

```
python-calculator/
│── calculator.py
│── history.txt
│── README.md
└── .gitignore
```

## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/labibaghani/python-calculator.git
```

2. Open the project folder:

```bash
cd python-calculator
```

3. Run the program:

```bash
python calculator.py
```
